# 源代码
源码文件一般使用项目名字作为前置，当前项目规定使用 *Mine* 作为前置名字
## 预定义宏
创建项目时会预定义项目DLL相关的宏，当前为：*UEEXAMPLES_API*，该宏用于对外公开的类和接口，特别是插件对外公开的类必要要有*UEEXAMPLES_API*
## 模块
逻辑模块使用目录分隔，有的喜欢区分公开或私有目录(public/private)区分头文件和定义文件，这个项目将头文件和源码放在一起
